SYSTEM REQUIREMENTS:
-------------------
- Python 3.8 or higher (tested with Python 3.12.4)
- Required packages: numpy, scikit-learn, matplotlib

INSTALLATION:
-------------
This program uses standard Python libraries available on all platforms.

Option 1 - Using Conda (What I used):
    conda install numpy scikit-learn matplotlib

Option 2 - Using pip:
    pip install numpy scikit-learn matplotlib

REQUIRED FILES:
--------------
- polynomial_curve_fitting.py (main program)
- train.dat (training data - 60 samples)
- test.dat (test data - 25 samples)

All three files must be in the same directory.

HOW TO RUN:
-----------
1. Open a terminal/command prompt
2. Navigate to the directory containing the files:
   cd /path/to/project/directory

3. Run the program:
   python polynomial_curve_fitting.py

   OR (if python3 is required on your system):
   python3 polynomial_curve_fitting.py

EXPECTED FILES GENERATED:
------------------------
- polynomial_fit.png (plot of training data and fitted polynomial)
